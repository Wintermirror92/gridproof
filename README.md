# GridProof ESS

> 중앙계약시장 ESS 공공사업 이행검증 AI  
> "제어하지 않는다. 검증한다."

🔗 Demo: https://gridproof-ess.streamlit.app/

---

## Data Notice

본 MVP는 **공개자료 기반 환경 분석 + 합성 로그 기반 ESS 이행검증 프로세스**를 구현한 시뮬레이션입니다.

- 실제 KPX 지령 로그를 사용하지 않았습니다.
- 실제 ESS BMS/PCS 운영 로그를 사용하지 않았습니다.
- 실제 사업자 실명·가입자 정보는 사용하지 않았습니다.
- 합성 데이터는 지령값, 실제출력, SoC, 통신결측, PCS/BMS 알람, 품질플래그, 기상특보 상황을 재현하기 위해 생성했습니다.
- AI는 부정수급, 고의 미이행, 법규 위반을 확정하지 않습니다.
- AI는 담당자 검토를 위한 위험도, 확인필요 구간, Action Card, 감사 리포트 초안을 제공합니다.

---

## MVP Scope

| 구분 | 내용 |
|---|---|
| 공개자료 | ESS 설비용량, 출력제어, 전력시장·정산 제도, 관련 기사·공고 |
| 합성 로그 | 지령값, 실제출력, SoC, 통신상태, 알람, 품질플래그 |
| 산출물 | 이행률, 위험점수, 점검 우선순위, Action Card, 감사 리포트 초안 |
| 실제 도입 필요 | KPX/VPP 지령 로그, ESS BMS/PCS 로그, 계량 데이터, 기상 API, 점검 이력 |

## Key Message

GridProof ESS는 ESS를 제어하지 않습니다.  
KPX 지령을 대체하지 않습니다.  
VPP/PMS와 경쟁하지 않습니다.  

GridProof ESS는 공공사업으로 설치·운영되는 ESS가  
**신청·설치·운영·지령 이행 단계에서 약속대로 작동했는지 검증하는 AI**입니다.
